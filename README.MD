## Instructions

1. Clone the repo
2. `cd` to the cloned repo (i.e.: `cd xwiki-test`)
3. 
    1. Run xwiki  
        `docker compose up -d`
    2. Stop xwiki  
        `docker compose down`

Then wait to docker containers to be ready and try [localhost:8080](http://localhost:8080)
